# stockmate-backend

Firebase Functions backend for invoice extraction using Google Vision API + Gemini API.

## 1) Install

```bash
npm install
```

## 2) Login and select Firebase project

```bash
firebase login
firebase use <your_firebase_project_id>
```

## 3) Set function secret

```bash
firebase functions:secrets:set GEMINI_API_KEY
```

## 4) Build and deploy

```bash
npm run build
firebase deploy --only functions
```

## 5) Local emulator

```bash
npm run serve
```

## Endpoint

- Function name: `invoiceExtract`
- Method: `POST`
- Auth: Firebase ID token in `Authorization: Bearer <token>`
- Body:

```json
{
  "imagePath": "invoice-uploads/<uid>/<file>.jpg",
  "supplierHint": "optional supplier name"
}
```

## Response

```json
{
  "draftId": "autoGeneratedFirestoreId",
  "draft": {
    "supplierName": "PT SUPPLIER",
    "receiptCode": "STRUK-001",
    "receiptDate": "2026-03-29",
    "note": "",
    "rows": [],
    "overallConfidence": 0.91
  }
}
```
